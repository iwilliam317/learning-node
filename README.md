# NodeJS

###### This project is created for studying NodeJS using MVC pattern and external modules.

## Modules Included

###### Express
###### EJS
###### Nodemon
###### Body Parser
###### Express Validator
###### Consign
###### MySQL

##Running Project

###### Open the shell
###### Access the directory of the project
###### Locate the app.js 
###### Run the code below:
```nodemon app ```
